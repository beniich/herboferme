## 2026-03-04 - Dashboard Query Consolidation
**Learning:** The dashboard route was making over 10 sequential database round-trips. By using MongoDB's `$facet` stage, we can group multiple counts and groupings for the same collection into a single database operation.
**Action:** Use `$facet` and `Promise.all` for all dashboard-style endpoints to minimize network latency between the app and the database.
